Ce document décrit les éléments codemeta utilisés par Software Heritage (SH) afin d'une part de fournir un schéma qui permette à SH de valider leurs exports/import et d'autre part de prévoir l'intégraitonde de ces éléments dans le format d'échange HAL-TEI pour couvrir le dépot de logiciel dans HAL.

La documentation actuelle de SH sur Codemeta se trouve sous https://docs.softwareheritage.org/devel/swh-deposit/metadata.html?highlight=atom
